# mayo2026 — Caminos alternativos al pivote de abril

Documentos de revisión y opciones estratégicas para el proyecto MCP tras leer el estado del repo en abril/mayo de 2026. No reemplazan a [PIVOTE_2026.md](../PIVOTE_2026.md), lo complementan ofreciendo cinco trayectorias distintas y una recomendación.

## Índice

1. [00-REVISION.md](00-REVISION.md) — Lectura crítica del estado actual: qué funciona, qué riesgos persisten, qué activos existen.
2. [OPCION-A-libro-lean.md](OPCION-A-libro-lean.md) — Libro corto (80-120 págs) como tarjeta de presentación + lead magnet.
3. [OPCION-B-medimind-saas.md](OPCION-B-medimind-saas.md) — Apuesta máxima: MediMind como SaaS multi-tenant para clínicas ES/LatAm.
4. [OPCION-C-consultoria-mcp-security.md](OPCION-C-consultoria-mcp-security.md) — Servicios productizados de auditoría/hardening MCP. Time-to-revenue corto.
5. [OPCION-D-curso-cohortes.md](OPCION-D-curso-cohortes.md) — Curso premium en vivo en español, 4 cohortes/año.
6. [OPCION-E-toolkit-oss.md](OPCION-E-toolkit-oss.md) — `mcp-audit`: scanner OSS + tier Cloud + Enterprise.
7. [RECOMENDACION.md](RECOMENDACION.md) — Síntesis: combo recomendado (C + A → E), roadmap 12 meses y métricas de decisión.
8. [ejecucion/](ejecucion/) — **Ejecución del combo C+A→E**: entregables accionables (sales pages, plantillas, outbound, landing, esqueleto del libro, spec OWASP MCP Top 10, roadmap mes a mes y dashboard de métricas).

## Cómo usar este material

- Leer en orden: revisión → opciones → recomendación.
- Ninguna opción es prescriptiva. Cada una se descarta o elige según las preguntas abiertas en [00-REVISION.md](00-REVISION.md) §4 (capital disponible, dedicación, red profesional, apetito de riesgo).
- Las opciones **se pueden combinar**. La recomendación principal lo hace.
- Antes de elegir cualquiera: ejecutar las 5 entrevistas de validación que el pivote de abril dejó pendientes.
