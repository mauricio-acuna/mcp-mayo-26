# Organizaciones objetivo - Semana 1

> Lista de partida para convertir en personas concretas. No contactar a
> empresas: buscar primero el rol adecuado y una senal personalizable.

## Prioridad de segmentos

1. **MCP real en produccion o publico.** Mejor senal para validar el dolor exacto.
2. **Healthtech / clinica con IA y datos sensibles.** Encaja con MediMind y FHIR.
3. **Fintech / insurtech regulada.** Buen proxy de compliance, DORA, ENS, SOC 2.
4. **Ciberseguridad / GRC / auditoria.** Buen proxy de comprador o prescriptor.
5. **Consultoras IA / plataforma.** Buen proxy de demanda agregada.
