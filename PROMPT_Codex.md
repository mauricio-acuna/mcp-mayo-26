# Prompt para Codex

Lee los siguientes archivos en este orden:

1. `HANDOFF.md`: Contiene el estado actual del proyecto, decisiones pendientes, y tareas priorizadas.
2. `README.md`: Proporciona una visión general del repositorio y detalles técnicos clave.
3. `PIVOTE_2026.md`: Describe la última decisión estratégica vigente.

**Objetivo**:
- Proponer un plan detallado para continuar el desarrollo del proyecto.
- Identificar cualquier dependencia o configuración faltante.
- Priorizar las tareas según el contexto proporcionado.

**Instrucciones**:
1. Analiza los archivos mencionados.
2. Proporciona un plan de acción claro y priorizado.
3. No realices cambios en el código sin confirmar el plan con el humano responsable.
